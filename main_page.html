About

About The Project

The product allows teachers to control computers in their classroom. With the help of the product, the teacher can perform many with only the flick of a finger.
The purpose of the product is to help teachers manage and monitor the classroom computers efficiently and simply. Without any need to intervene.

Product goals:
- Fast (the product works fast, better than most if not all)
- Does not require knowledge for operation
- Safe (the product ensures that there will be no hacking of students' computers by a malicious person from the school network)

Installation

Now we will go over the steps to install "Class Controller"

  • step 1:
    First press the button to download the version you want (Server/Client)
  • step 2:
    Now lets extract the installer with an extraction software (like winrar)
  • step 3:
    after the extraction, double-click the installer and follow its instructions
  • step 4: (very important!!)
    double click the __init__.exe file in the directory "C:\Program Files (x86)\Class Controller"

How To Use

Server For the server, you are welcome to explore the different features when you setup a client. list of features:

  • get info
  • transfer files
  • execute commands
  • video share
  • disable wi fi
  • turn off client computers
  • browse client computer files


Client

For the client, the software should run every time the computer starts(to do this please add the software to autostart) and get commands from the server. this happens automatically without any input from the user.
